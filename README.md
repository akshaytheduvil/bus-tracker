# 🚌 ENTE BUS

**ENTE BUS** is a simple, fast, and mobile-friendly bus timetable web application designed to help passengers quickly check bus timings between **Pukayoor, Chemmad, Kunnumpuram**, and nearby locations.

The project focuses on providing an easy-to-use interface with instant bus search, responsive design, and offline-friendly performance.

---

## 🌟 Features

- 📱 Mobile-first responsive design
- 🔍 Find the next available bus instantly
- 🕒 Live timetable lookup
- 🚌 Multiple route support
- 🌙 Modern and clean UI
- ⚡ Fast loading
- 📥 Download timetable image
- 🌐 Works on all modern browsers
- 🇮🇳 Malayalam-friendly interface
- 💻 Easy to customize

## 🚍 Supported Routes

Current routes include:

- Pukayoor → Chemmad
- Chemmad → Pukayoor
- Pukayoor → Kunnumpuram
- Kunnumpuram → Pukayoor

Additional routes can be added easily by updating the bus data file.
